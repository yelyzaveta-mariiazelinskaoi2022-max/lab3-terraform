# Lab 3 - Terraform AWS Infrastructure

Deploying Apache web server on AWS EC2 using Terraform.
Variant 08 - Student: Zelinska Liza

## Prerequisites
- AWS CLI configured
- Terraform >= 1.0
- S3 bucket for remote state already created

## Usage

### Initialize
terraform init

### Preview changes
terraform plan

### Deploy infrastructure
terraform apply

### Destroy infrastructure
terraform destroy
